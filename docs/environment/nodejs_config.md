## node.js的安装
  
vuepress是基于node.js语言的，因此需要到[node.js官方网站](http://nodejs.cn/download/)去下载，并记住它的安装路径。

## 环境变量的配置

首先打开命令提示符（通过快捷键`Windows + R`，并输入 cmd 打开），在命令行中输入 npm ，看看有没有这个命令。如果没有的话则需要进行环境变量的配置。
